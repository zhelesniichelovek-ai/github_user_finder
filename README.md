# Currency Converter — Конвертер валют

Приложение с графическим интерфейсом (GUI) для быстрого пересчета сумм между различными мировыми валютами с использованием актуальных рыночных курсов.

## Автор
* **Мартынов Сергей Константинович**

## Как получить API-ключ
Проект использует открытый и бесплатный API-сервис **ExchangeRate-API** (эндпоинт: `https://er-api.com`).
* Для базовых запросов и получения актуального списка курсов индивидуальный секретный API-ключ **не требуется**. 
* Запросы выполняются к публичному API напрямую методом `GET`. Сервис обновляет данные каждые 24 часа.

## Инструкция по запуску проекта
1. Убедитесь, что на компьютере установлен Python версии 3.8 или выше.
2. Программа использует встроенный графический движок `tkinter`. Необходимо доустановить только модуль `requests` для работы с сетью:
   ```bash
   pip install requests
   ```
3. Запустите скрипт из папки проекта:
   ```bash
   python windows.py
   ```

## Возможности приложения и работа с данными
* **Валидация:** Поле суммы строго проверяется регулярными выражениями. Разрешены только положительные числовые значения больше нуля. При некорректном вводе программа выдает предупреждение.
* **Локальный кэш (JSON):** История всех успешных операций автоматически записывается и подгружается из файла `history.json`.
* **Отказоустойчивость:** Если у пользователя отсутствует подключение к интернету или сервер API недоступен, программа не упадет, а автоматически переключится на заложенные константные значения курсов валют.
