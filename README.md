# GitHub User Finder

Программа с графическим интерфейсом (GUI) для поиска профилей на GitHub и сохранения их в локальный список избранного.

## Автор
* **Мартынов Сергей Константинович**

## Инструкция по запуску проекта
1. Убедитесь, что у вас установлен Python версии 3.x.
2. Установите необходимую библиотеку для работы с сетью (`requests`), запустив команду в терминале:
   ```bash
   pip install requests
   ```
3. Запустите приложение командой:
   ```bash
   python windows.py
   ```

## Как использовать API
Приложение взаимодействует с официальным публичным REST API GitHub.
* **Эндпоинт:** `https://github.com{username}`
* **Метод:** `GET`
* **Формат ответа:** JSON

## Примеры использования
1. Введите в текстовое поле корректный логин (например, `vanopy`).
2. Нажмите кнопку **"Найти пользователя"**. В случае успеха на экране отобразится логин, имя и ссылка на профиль.
3. Нажмите кнопку **"Добавить в избранное"** — данные отформатируются и запишутся в файл `favorites.json`, а также отобразятся в нижнем списке приложения.
4. Попробуйте ввести недопустимые символы (например, кириллицу) или оставить поле пустым, чтобы проверить работу валидации.
